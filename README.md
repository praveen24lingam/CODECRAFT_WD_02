# Stopwatch Web Application | Code Craft

## Description
An interactive stopwatch allowing users to start, pause, reset, and record lap times with a clean and responsive interface.

## Features
- Start / Pause toggle button
- Reset functionality
- Lap recording
- Responsive and modern UI

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)

## Files
- `index.html`
- `style.css`
- `script.js`

## How to Run
1. Open `index.html` in your browser.
2. Click start to begin timing, pause to pause, reset to clear, and lap to record lap times.
